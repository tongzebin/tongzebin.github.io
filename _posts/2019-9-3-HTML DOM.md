### HTML DOM

1. 什么是 DOM？

DOM 是 W3C（万维网联盟）的标准。
DOM 定义了访问 HTML 和 XML 文档的标准

2. 什么是 HTML DOM？
    ***HTML DOM 是：***
    - HTML 的标准对象模型
    - HTML 的标准编程接口
    - W3C 标准
3. 常用命令
    - document.getElementById()	通过元素id选择元素
    - document.getElementByTagName()	通过标签名选择元素
    - document.getElementByClassName()	通过类名选择元素
    - document.CreateElement()	创建元素
    - 	      .removeclid()	删除元素
    -         .appendchild()	添加元素
    -         .replacechild()	替换元素
